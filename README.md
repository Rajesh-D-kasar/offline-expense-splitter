# Offline Expense Splitter

This is a simple offline expense splitter mini project made with only HTML, CSS and JavaScript.

## Features

- Add friends
- Add expense name and amount
- Select who paid the expense
- Select friends for split
- Show total expense
- Show each friend's paid amount, share and balance
- Show final settlement
- Delete friends and expenses
- Reset all data
- Save data in browser using localStorage

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in any browser. Internet is not required.

## Project Explanation

The app stores friends and expenses in JavaScript arrays. When a new friend or expense is added, the app recalculates balances and final settlement. Data is saved in browser localStorage, so it remains available even after refreshing the page.
